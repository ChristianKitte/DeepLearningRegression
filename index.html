In diesen Bereich wird ein Überblick über die zugrunde liegenden Daten dieser Aufgabe gegeben. Bei dem Training werden N = 5, 10, 20, 50 und 100 gleichverteilte Datensätze (keine Normalverteilung) als x-Werte jeweils neu gezogen. Es handelt sich also nicht um reduzierte Daten einer einzelnen Ziehung.

Die X-Werte dienen als Feature der Regression. Die berechneten Werte entsprechen den dazugehörigen Labels. Zusätzlich wurden den Labels ein normalverteiltes Rauschen mit wahlweiser Standardabweichung von 0,1 bzw. 0,5 hinzugefügt (Mittelwert = 0).

Als Generatoren kamen für die gleichverteilten Features rechneronline.de und für das Rauschen statologie.de zum Einsatz.

Das Ausgewählte DataSet wird bei den weiteren Versuchen zur Regression als Basis dienen. Hierdurch können Rückschlüsse zur Auswirkung der Datenqualität beim Trainieren mit einem neuronalen Netz gezogen werden.

Mit den im lnken Reiter eingestellten Dataset kann an dieser Stelle ein feed-forward neural Network (FFNN) trainiert werden. Das Ziel ist hierbei eine Regression der reellwertigen Funktion: y(x) = (x+0.8) * (x-0.2) * (x-0.3) * (x-0.6) im Wertebereich [-1,+1].
