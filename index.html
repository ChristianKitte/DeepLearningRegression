Hilfsfunktionen

Mit einem feed-forward neural Network (FFNN) soll eine Regression der reellwertigen Funktion: y(x) = (x+0.8) * (x-0.2) * (x-0.3) * (x-0.6) im Wertebereich [-1,+1] durchgeführt werden. Zum Erzeugen der Trainingsdaten sollen N zufällige, gleich-verteilte x Werte (mit N= 5, 10, 20, 50, 100) aus dem Intervall [-1,+1] (keine Normalverteilung für die x-Werte) erzeugt und dazu y(x) berechnet werden. Zum Verrauschen wird dem Funktionswert ein normal-verteiles Rauchen (Gaussian Noise) mit einer Varianz von entweder 0.1 oder 0.3 hinzugefügt. Im Abschnitt Dokumentation finden sie eine ausführliche Erklärung hierzu.
