In diesen Bereich wird ein Überblick über die zugrunde liegenden Daten dieser Aufgabe gegeben. Für das Training wurden N = 5, 10, 20, 50 und 100 gleichverteilte Datensätze (keine Normalverteilung) als Feature gezogen. Es handelt sich hierbei jeweils um separate Ziehungen, also nicht um reduzierte Daten einer einzelnen Ziehung.

Die anhand der obigen Formel berechneten Werte entsprechen den dazugehörigen Labels. Zusätzlich kann den Labels ein normalverteiltes Rauschen mit Standardabweichung von Wahlweise 0,1 bzw. 0,5 hinzugefügt (Mittelwert = 0) werden.

Als Generatoren kamen für die gleichverteilten Features rechneronline.de und für das Rauschen statologie.de zum Einsatz.

Das Ausgewählte DataSet wird bei den weiteren Versuchen zur Regression als Basis dienen. Hierdurch können Rückschlüsse zur Auswirkung der Datenqualität beim Trainieren mit einem neuronalen Netz gezogen werden.

Oben rechts sehen Sie, welches DataSet aktuell ausgewählt und geladen wurde.

Mit den im lnken Reiter eingestellten Dataset kann an dieser Stelle ein feed-forward neural Network (FFNN) trainiert werden. Das Ziel ist hierbei eine Regression der reellwertigen Funktion: y(x) = (x+0.8) * (x-0.2) * (x-0.3) * (x-0.6) im Wertebereich [-1,+1].
